  A smart team task management platform with AI-powered prioritization

<hr>

<h2>About The Project</h2>
<p>
TaskMind AI is a modern SaaS-based task management system designed for teams, colleges, and organizations.
It helps users manage tasks efficiently using role-based access and AI-driven prioritization.
</p>

<ul>
  <li>Team Members can view and complete tasks</li>
  <li>Teachers can assign and manage tasks</li>
  <li>Mentors can monitor team performance</li>
</ul>

<hr>

<h2>Features</h2>

<h3>Authentication & Roles</h3>
<ul>
  <li>Secure login & signup system</li>
  <li>Role-based access (Team Member, Teacher, Mentor)</li>
</ul>

<h3>Invite System</h3>
<ul>
  <li>Faculty can invite team members via email</li>
  <li>Unique invite link with token</li>
  <li>Users set their own password securely</li>
</ul>

<h3>Task Management</h3>
<ul>
  <li>Create, assign, and manage tasks</li>
  <li>Set deadlines and difficulty levels</li>
  <li>Track task status (Pending / Completed)</li>
</ul>

<h3>AI Task Prioritization</h3>
<ul>
  <li>Smart priority calculation based on:</li>
  <ul>
    <li>Deadline</li>
    <li>Task difficulty</li>
    <li>User productivity</li>
  </ul>
  <li>Priority levels: High / Medium / Low</li>
</ul>

<h3>Productivity Tracking</h3>
<ul>
  <li>Completion percentage</li>
  <li>Performance insights</li>
</ul>

<h3>Team Management</h3>
<ul>
  <li>Create and manage teams</li>
  <li>Add members via invite system</li>
</ul>

<hr>

<h2>Tech Stack</h2>

<ul>
  <li><b>Frontend:</b> React.js, Tailwind CSS</li>
  <li><b>Backend:</b> Node.js, Express.js</li>
  <li><b>Database:</b> MongoDB</li>
  <li><b>Authentication:</b> JWT, Bcrypt</li>

<h2>How It Works</h2>

<ol>
  <li>Faculty invites a team member</li>
  <li>User receives an invite link</li>
  <li>User sets password and joins team</li>
  <li>Teacher assigns tasks</li>
  <li>Team member completes tasks</li>
  <li>AI prioritizes tasks and tracks productivity</li>
</ol>
